# Escalares (Scalar Types)

- Representam um único valor contido dentro de uma escala conhecida.
- Permitem a comparação direta entre valores.

## Tipos

- [inteiro](./inteiros.md) (integer) 
    - ex: `5`
- flutuante (floating point / float) 
    - ex: `77.7`
- booleano (bool) 
    - ex: `true`, `false`
- caractere (char)  
    - ex: `'g', '和平
','❤️‍🔥'`
